How to Hack Facebook, Twitter, or Gmail

Djangelic

Hacking someones Facebook or Twitter accounts is not hard, thanks to Facebook and Twitter. Why is this? Because Facebook and Twitter have a GIANT hole in them. It's called cookies! When you login to Facebook, a little file is created that allows you to remain logged in, so you're not constantly logging in again. The thing is that little file has to constantly be checked by facebook. Which means that if you have the right program, you can intercept that file. Now, here is the limitation. You have to be on the same WIFI as the other person. So this will work on dorms, and in library's, but if your in a different location (or different network), it wont work. Read on to see how its done:

UPDATE: Please note, that sniffing public networks for data that isn't yours, is illegal. For more information on this, please read this blog post.

It's actually very simple. You need 3 things.

1. Firefox - HERE

2. A program to sniff the internet - HERE

3. A script called firesheep - HERE

Start by installing Firefox. That should be simple enough and shouldnt need a tuturial. Then download and install WinPCAP. It allows your computer to sniff packets of information on the network. that one should also be quite easy. Lastly install firesheep. That might be a bit more complicated, so I've included screenshots.

click on the windows link for the download, and it will take you to this site:

Then click on the file that you see in the middle. Firefox will ask you what you want to do with the file:

Choose the Open With option, and choose firefox from the list. After it downloads, it will want to install itself on firefox.

Wait until it allows you to Install Now, and click on it.

After it's installed, you need to enable it, to do so, go to View > Sidebar > Firesheep

you will then see the sidebar, which is how you control firesheep. Click on start capture to start sniffing the packets in the local network:

That's pretty much it. Now you just have to wait for people to use their accounts, and you will start seeing them on the left hand corner. To hack their account, simply doubleclick on their picture.

Now, I'm telling you this, so you see how easy it is to hack your account. Always be careful when accessing accounts in public places. this includes your banking info, your social networks, and even email. I don't want to make you paranoid, but remember, sensitive things should be done in your home, not out in public. Now, if your scared, don't worry, there is a way to protect yourself from this. Techcrunch created a wonderful article that shows you how to protect yourself, and you can access it here: http://techcrunch.com/2010/10/25/firesheep/

The original post by the creator of this hack can be accessed here: http://codebutler.com/firesheep
